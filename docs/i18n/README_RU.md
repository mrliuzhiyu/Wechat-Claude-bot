## Возможности

- **Удалённое управление через WeChat** — Управляйте локальным Claude Code напрямую через сообщения WeChat, не нужно открывать терминал
- **Работает после закрытия WeChat** — Благодаря серверному long polling, сообщения ставятся в очередь даже после закрытия приложения WeChat. Откройте WeChat снова — и увидите ответы. Бот работает 24/7 без перерывов
- **Прогресс в реальном времени** — Получайте уведомления о ходе работы: чтение файлов, написание кода, выполнение команд
- **Полные возможности работы с кодом** — Claude Code имеет полные права: чтение/запись файлов, выполнение команд, поиск кода, установка пакетов
- **Изоляция сессий пользователей** — Каждый пользователь WeChat получает независимую сессию с непрерывным контекстом
- **Поддержка голосовых сообщений** — Поддержка голосовых сообщений WeChat (требуется преобразование голоса в текст)
- **Умное разделение сообщений** — Длинные ответы автоматически разделяются по границам блоков кода
- **Конвертация Markdown** — Автоматическое преобразование Markdown-вывода Claude в формат WeChat
- **Автоматическое переподключение** — Автоматический повторный вход при истечении сессии, сохранение токена
- **Контроль параллелизма** — До 3 одновременных задач, остальные автоматически ставятся в очередь
- **Корректное завершение** — Поддержка Ctrl+C для безопасного выхода с автоматической очисткой

## Принцип работы

```
┌──────────┐         ┌──────────────────┐         ┌─────────────┐
│  WeChat   │ ─сообщ.─▶│  iLink Bot API   │ ─опрос──▶│ Локальный   │
│ (телефон) │ ◀─ответ── │ (weixin.qq.com)  │ ◀─отпр.── │ Bot(Node.js)│
└──────────┘         └──────────────────┘         └──────┬──────┘
                                                         │
                                                         │ Вызов CLI
                                                         ▼
                                                   ┌───────────┐
                                                   │ Claude Code│
                                                   │ (локальный)│
                                                   └───────────┘
```

1. Бот получает сообщения через WeChat iLink Bot API (long polling)
2. Пересылает сообщения в локальный Claude Code CLI (режим stream-json)
3. В реальном времени анализирует вызовы инструментов Claude Code, отправляя прогресс в WeChat
4. После завершения форматирует и отправляет результат обратно в WeChat

## Быстрый старт

### Требования

- **Node.js** >= 18
- **Claude Code CLI** установлен глобально (`npm install -g @anthropic-ai/claude-code`)
- **Аккаунт WeChat**

### Установка

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/mrliuzhiyu/Wechat-Claude-bot.git
cd Wechat-Claude-bot

# 2. Установите зависимости
npm install

# 3. (Опционально) Настройте рабочую директорию
cp .env.example .env
# Отредактируйте .env, установив CLAUDE_CWD на путь к вашему проекту

# 4. Запустите бота
npm start
```

### Первое подключение

1. После запуска в терминале отобразится QR-код
2. Откройте WeChat → Отсканируйте QR-код
3. Подтвердите подключение в WeChat
4. Когда увидите `✅ Подключено!`, бот готов к работе
5. Просто отправьте сообщение боту в WeChat

> После первого входа токен сохраняется автоматически. Повторное сканирование не требуется (кроме случаев истечения токена).

## Руководство по использованию

### Основное использование

Описывайте свои потребности на естественном языке в WeChat. Claude Code выполнит автоматически:

```
Вы: Покажи структуру проекта
Bot: 🤖 Получено, обрабатываю...
Bot: 🔍 Поиск файлов: **/*
Bot: Структура проекта:
     ├── src/
     │   ├── components/
     │   ├── pages/
     │   └── utils/
     ├── package.json
     └── README.md
```

### Что умеет Claude Code

Через сообщения WeChat вы можете:

- **Чтение кода** — «Покажи содержимое src/app.js»
- **Написание кода** — «Создай функцию форматирования даты в utils»
- **Изменение кода** — «Измени цвет фона компонента App на синий»
- **Выполнение команд** — «Запусти npm test и покажи результаты»
- **Поиск кода** — «Найди все места, где используется useState»
- **Установка пакетов** — «Установи axios и lodash»
- **Отладка** — «Почему сборка не проходит? Проверь»
- **Код-ревью** — «Проверь последние изменения на наличие проблем»
- **Git-операции** — «Сделай коммит с сообщением "исправление бага авторизации"»

### Прогресс в реальном времени

При выполнении операций Claude Code вы получаете обновления:

```
📖 Чтение файла: src/app.js
✏️ Редактирование файла: src/utils.js
⚡ Выполнение команды: npm test
🔍 Поиск файлов: **/*.ts
🔍 Поиск содержимого: handleClick
📝 Создание файла: src/helper.js
📋 Планирование задач
```

### Обработка длинных сообщений

Когда ответ Claude Code превышает 4000 символов, сообщения умно разделяются:

- Приоритет — разделение на границах блоков кода
- Затем — на пустых строках
- Каждый фрагмент помечен номером, например: `(прод. 2/3)`

## Часто задаваемые вопросы

### В: При запуске «команда claude не найдена»

Убедитесь, что Claude Code CLI установлен глобально:

```bash
npm install -g @anthropic-ai/claude-code
```

Проверьте командой `claude --version`.

### В: QR-код отображается некорректно

Если терминал не поддерживает Unicode, QR-код может отображаться неправильно. В логе запуска есть URL — откройте его в браузере для сканирования.

### В: Что если токен истёк?

Бот автоматически обнаруживает истечение токена и показывает новый QR-код. Ручных действий не требуется.

### В: Могут ли несколько человек использовать одновременно?

Да. Каждый пользователь WeChat получает независимую сессию. Поддерживается до 3 одновременных запросов; остальные автоматически ставятся в очередь.

### В: Запрос прервался по таймауту

Таймаут по умолчанию — 5 минут на запрос. Для сложных задач разделите их на меньшие шаги — например, сначала попросите Claude исследовать структуру проекта, затем выполните конкретные операции.

### В: Поддерживается ли отправка изображений/файлов?

В настоящее время поддерживаются только текстовые и голосовые сообщения (с включённым преобразованием в текст). Изображения, видео и файлы пока не поддерживаются.

### В: Насколько это безопасно?

- Бот работает локально на вашей машине — код не проходит через сторонние серверы
- Claude Code работает в режиме `bypassPermissions` с полным доступом к файлам и командам
- Учётные данные хранятся локально в `.state/` с правами только для владельца
- `.env` включён в `.gitignore` и не попадёт в Git

> **Внимание**: Поскольку Claude Code имеет полные права, убедитесь, что только доверенные лица могут отправлять сообщения боту.
